KALKULATOR C++

PROGRAM KALKULATOR SEDERHANA DENGAN DUA INPUTAN BILANGAN. ADA BEBERAPA JENIS OPERATOR DALAM PROGRAM KALKULATOR INI DIANTARANYA PERKALIAN, PEMBAGIAN, PERTAMBAHAN, DAN PENGURANGAN.

SIMBOL OPERATOR

x PERKALIAN
: PEMBAGIAN
+ PERTAMBAHAN
- PENGURANGAN

INSTALL

1. apt-get upgrade && apt-get update

2. apt-get install git

3. apt-get install gcc

4. git clone https://github.com/imsikunyuk/KALKULATOR.git

5. cd KALKULATOR/cod

6. gcc -o KALKULATOR kalkulator_sederhana.cpp

7. ./KALKULATOR
